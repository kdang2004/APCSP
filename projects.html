Projects

Here is a project from my summer course from Cal Poly's EPIC program. I made a Rube Goldberg Project using an Arduino. I had to build the actual machine, and then code the Arduino so that the ball would get knocked down, fall into the boat, the fan would move, and then set off a motion sensor that made noise. This was really fun because I learned about Arduino and I got to make modifications to the project.
